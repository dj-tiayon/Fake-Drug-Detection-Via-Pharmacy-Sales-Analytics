# 📊 Dashboard Preview

This folder contains screenshots of the Power BI dashboard pages developed for the **Fake Drug Detection via Pharmacy Sales Analytics** project.

The dashboard was designed to help non-technical health stakeholders monitor suspicious pharmaceutical activity, investigate supplier risks, and identify potential counterfeit drug distribution patterns.

---

## 📄 Dashboard Pages

### 1️⃣ Drug Risk Overview
Provides a high-level summary of suspicious pharmaceutical activity, key risk indicators, and overall anomaly trends across the supply chain.

`drug-risk-overview.png`

---

### 2️⃣ Risk Investigation
Explores suspicious sales behavior, supplier-related risks, expiry patterns, and anomaly drivers in greater detail.

### 3️⃣ Supplier Analysis
Provides a detailed investigation page for analyzing supplier-specific suspicious activity and high-risk transaction patterns.
